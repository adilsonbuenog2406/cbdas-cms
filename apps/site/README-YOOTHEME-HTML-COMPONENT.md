# HTML Component YOOtheme - IDASAN/CBDAS

Arquivo principal: `idasan-cbdas-yootheme-html-component.html`.

## Como usar

1. Envie a pasta `yootheme-html-assets/cbdas-dist/` para `/public_html/images/cbdas-dist/`.
2. Abra a página no YOOtheme Pro.
3. Adicione um elemento/componente `HTML`.
4. Cole todo o conteúdo de `idasan-cbdas-yootheme-html-component.html` no campo HTML.
5. Salve e limpe o cache.

## Observações

- Este HTML foi capturado do DOM real renderizado por `dist/index.html` via Chrome headless.
- O CSS compilado do `dist/assets/index-BH_GD9eL.css` foi embutido no próprio HTML.
- Os scripts React originais não foram incluídos; o arquivo é um snapshot estático visual com links e rolagem suave.
- O JS embutido é isolado em `.cbdas-yootheme-html-component`.
